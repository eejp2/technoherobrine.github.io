Tags do HTML (Quais são e o que fazem)

Se você já mexeu com HTML sabe que existem muitas tags, mas nem todo mundo sabe o que elas querem dizer. Então, hoje vamos aprender pra que algumas tags servem.

body
p
span
h1, h2, h3, h4, h5 e h6
hr
center
